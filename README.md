# PE Mobile - SimPro Mobile Replacement

**A custom mobile app for Platinum Electricians that replaces SimPro mobile with a faster, more intuitive interface.**

## 🚀 Features

### ⏱️ Time Tracking
- **Start/stop timers** for jobs
- **Real-time display** of hours worked  
- **Automatic logging** to SimPro job records

### 📝 Job Management
- **Today's schedule** for each technician
- **Job notes** saved directly to SimPro
- **Real job data** pulled from SimPro API

### 🛒 Purchase Orders
- **Create PO's** directly from job sites
- **Supplier selection** (Haymans, Rexel, BLA)
- **Real PO numbers** (4xxxxx format) 
- **Links to specific jobs**

### 📷 Documentation
- **Photo uploads** for job records
- **Bulk photo handling**
- **Auto-tagging** to correct jobs
